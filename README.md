## components
My version does not use a status led. Do the led in the drawing does not need to be there

* Arduino Nano ATmega328
* OLED 128x64 I2C Display (128x64)
* 2 Controllers (10k potentiometer)
* Switch
* 1 resistor: 10K Switch
